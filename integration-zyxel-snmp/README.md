## Quick Start

```bash
# Build the image
docker build -t integration-zyxel-snmp:latest .

# Run the container
docker run -d \
  --name integration-zyxel-snmp \
  --hostname "$(hostname)" \
  --network rapax-dev-network \
  -v "$RAPAXHOME/lib:/opt/rapax/lib:ro" \
  -v "$RAPAXHOME/etc:/opt/rapax/etc:ro" \
  -v "$RAPAXHOME/logs:/opt/rapax/logs" \
  -v /etc/machine-id:/etc/machine-id:ro \
  -e REDIS_HOST=rapax-redis \
  -e "REDIS_PASSWORD=$REDIS_PASSWORD" \
  integration-zyxel-snmp:latest
```
